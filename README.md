# Hi there, I'm Asim 👋

### Senior Frontend Engineer | Shopify Developer | React & TypeScript Specialist

I’m a Senior Frontend Engineer with 5+ years of experience building scalable, performant, and user-focused web experiences. I specialize in modern frontend development, Shopify commerce, and production-ready UI systems using JavaScript, TypeScript, React, Vue.js, Liquid, and Tailwind CSS.
